# 开发准备

# 所需环境

1.JDK 1.8+

2.MySQL 5.5.0+

3.Redis 3.0+

4.Maven 3.0+

5.Node v10+

6.Vue-cli 3.0+

# 在使用该系统前，你还需要做如下准备

  1. 给 idea (opens new window) (opens new window)或者 eclipse (opens new window) (opens new window)安装 lombok 插件，我们用它可以省略get，set 方法，可以使代码更简洁， 具体查看 lombok教程(opens new window) (opens new window)
  2. 了解MapStruct，项目用到了它的映射实体，如果你不熟悉可以查看：熟悉MapStruct(opens new window) (opens new window)
  3. 你需要有 Spring boot 的基础，推荐教程 Spring Boot 2.0 学习(opens new window) (opens new window)
  4. 你需要有 Vue (opens new window) (opens new window)的基础，推荐入门视屏教程 网易云课堂 (opens new window)
  5. 熟悉git使用流程，推荐博客带你了解实际工作中git的使用流程 (opens new window)
  6. 从GitHub上拉取前后端项目以及数据库GitHub地址 (opens new window)
  7. 本地安装Redis，推荐教程 Redis Windows 64位下安装Redis详细教程 (opens new window)

# 5.启动Redis服务

  redis:
    #数据库索引
    database: 7
    host: 127.0.0.1
    port: 6379
    password:
    #连接超时时间（ms）
    timeout: 5000
    # 自定义redis默认过期时间（单位：时）
    expire-time: 24
    jedis:
      pool:
        # 连接池最大连接数（使用负值表示没有限制）
        max-active: -1
        # 连接池最大阻塞等待时间（使用负值表示没有限制）
        max-wait: -1

  servlet:
    multipart:
      # 开启 multipart 上传功能
      enabled: true
      # 文件写入磁盘的阈值
      file-size-threshold: 2KB
      # 最大文件大小
      max-file-size: 200MB
      # 最大请求大小
      max-request-size: 215MB

# 前端安装及启动

# 1. 用git克隆前端项目到本地

前端git地址：https://github.com/sanyueruanjian/smpe-admin-web

git clone https://github.com/sanyueruanjian/smpe-admin-web

# 2. 安装项目依赖

  1. 用编译器导入项目

  2. 安装依赖

    2.1. 建议使用npm安装依赖，使用cnpm安装会 出现奇怪的问题，可以用registry指向淘宝镜像解决npm下载速度慢

# 3. 安装依赖完成后，启动前端项目：npm run dev

# 4.打开浏览器，输入：(http://localhost:8013)
