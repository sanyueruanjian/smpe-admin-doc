# 版本迭代

# v1.0.0-release

# 新增功能

1.新增操作日志和异常日志查看，详情参看@log注解的使用

2.新增任务调度，管理定时任务

# bug修复

1.缓存方案升级，详情参看官方文档

2.修复数据权限问题(解决水平越权和垂直越权问题)

# 注意事项

smpe稳定版v1基本不会再新增太多功能，后期主要做优化和解决bug，v1最新版本参看smpe分支"release/v1"
